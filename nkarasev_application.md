**[ НАЗВАНИЕ ПРОЕКТА ]**

"Угадай кто?"

**[ ПРОБЛЕМНОЕ ПОЛЕ ]**

Сейчас многим из нас приходится работать с кем-то в группе. И в каждом коллективе обязательно будет человек, который окажется одиноким из-за того, 
что он очень стеснительный. Или, общаясь много лет, группа продолжает существовать в формате других маленьких компаний. Наш проект помогает сплочению коллектива. 
У всех будет возможность поговорить со всеми и ,что точно добавит уверенности в себе, анонимно. 

Проблемы:

* Сложности социальной адаптации новопоступивших.

* Несплоченность коллективов. 

* Ощущения отсутствия единомышленников в группе.

**[ ЗАКАЗЧИК / ПОТЕНЦИАЛЬНАЯ АУДИТОРИЯ ]**

Новопоступившие лицеисты, состоящие в одной группе. 

**[ АППАРАТНЫЕ ТРЕБОВАНИЯ ]** 

Продукт разрабатывается под IOS.

Версия IOS 12 и выше, 500 мб на гаджете

**[ ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ ]**

Программный продукт будет предоставлять следующие возможности:

* Прохождение пользователями психологического теста, регистрация польщователей и выдача им индивидуальных ключей для входа. 

* Разбиение пользовталей по чатам по результатам психологическиого теста

* Возможность у куратора менять собеседников вручную

* Ананомное общение пользователей

* Смена собеседника через определнный промежуток врмени

* Через определенный промежуток времени пользовтели переходят из формата диалога на групповое общение

* Получение поощрений за активность

* Возможность отправлять отзыв


**[ ПОХОЖИЕ / АНАЛОГИЧНЫЕ ПРОДУКТЫ ]**

Похожие: 

* Badoo. Отличия от нашего приложения: Badoo приложение для знакомств, в котором пользователи общаются не анонимно. В нашем приложении общаться будет класс только между собой и анонимно.  

* Tinder. Аналогично Badоо
